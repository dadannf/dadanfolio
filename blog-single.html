Install Software Asterisk dan Tanam Extention Asterisk Di Zoiper

  • Dadan Nuh Faturahman
  • Web Design

Asterisk adalah perangkat lunak gratis dan sumber terbuka yang dirancang untuk membuat aplikasi komunikasi, seperti sistem telepon IP, Gateway VoIP, server konferensi, dan solusi khusus lainnya untuk perusahaan atau organisasi Anda. Itulah mengapa ini sangat digunakan oleh banyak instansi, pemerintah, dan perusahaan. ZoiPer adalah perangkat lunak telepon (softphone), yang memungkinkan Anda melakukan obrolan, suara, panggilan video, dll.

Konfigurasi Software Asterisk di VM Debian 11

  1. Buka VirtualBox dan ubah pengaturan jaringan menjadi Bridged Adapter dengan interface WLAN (Wi-Fi) sebagai bridge.
  2. Buka Debian dan buka konfigurasi menggunakan perintah: #nano /etc/network/interfaces. Kemudian ubah eth0 menjadi DHCP.
  3. Cek IP Address dengan perintah: #ifconfig eth0.
  4. Cek koneksi internet dengan perintah: ping google.com.
  5. Masuk ke konfigurasi repository dengan perintah: #nano /etc/apt/source.list, lalu tambahkan repository internet dari Debian 8 yang sesuai dengan jenis Linux dan versinya.
  6. Update repository dengan perintah: #apt-get update.
  7. Install Asterisk menggunakan perintah: #apt-get install asterisk.
  8. Buka konfigurasi sip.conf menggunakan perintah: #nano /etc/asterisk/sip.conf, lalu tambahkan script seperti berikut: 
    [111] ; Nomor telepon
context = nama_konfigurasi
type = nama_grup
username = nama_kontak
secret = password_nomer_telepon
host = dynamic ; Gunakan dynamic agar lebih fleksibel
  9. Buka konfigurasi extensions.conf dengan perintah: #nano /etc/asterisk/extensions.conf, lalu tambahkan script konfigurasi sesuai kebutuhan.
  10. Restart Asterisk dengan perintah: #/etc/init.d/asterisk restart.

Tanam Extention Asterisk pada Zoiper

  1. Install aplikasi Zoiper di smartphone 1.
  2. Buka aplikasi dan login menggunakan akun yang telah dibuat di VOIP Server:
    • Username: nomor_telepon@IP_Address
    • Password: secret dari konfigurasi VOIP Server
  3. Masukkan IP Address dari VOIP Server, lalu klik Next.
  4. Langsung skip saja langkah selanjutnya.
  5. Pilih protokol SIP, kemudian klik Finish.
  6. Ulangi langkah yang sama di smartphone 2.

Proses instalasi dan konfigurasi selesai.
